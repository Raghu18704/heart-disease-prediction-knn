# Heart Disease Prediction using KNN

This project predicts whether a person has heart disease using the K-Nearest Neighbors (KNN) algorithm.  
It includes a complete Flask web application with HTML and CSS frontend.

## Features
- Machine Learning model using KNN
- Feature scaling with StandardScaler
- Flask backend
- Clean HTML + CSS UI
- Real-time prediction

## Dataset
UCI Heart Disease Dataset(KAGGLE)

## Algorithm Used
- K-Nearest Neighbors (KNN)

##  Tech Stack
- Python
- Scikit-learn
- Flask
- HTML
- CSS
